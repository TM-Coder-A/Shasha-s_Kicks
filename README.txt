SHASHA'S KICKS WEBSITE WITH ADMIN LOGIN

Files included:
- index.html        Main website
- admin.html        Admin login and dashboard
- style.css         Website and admin styling
- product-store.js  Shared product/settings storage
- app.js            Website product display and settings updater
- admin.js          Admin login, add/edit/delete products
- assets/           Flyer/product image folder

ADMIN LOGIN
Open admin.html in your browser.

Default login:
Username: admin
Password: Shasha@123

WHAT THE ADMIN PAGE CAN DO
- Add products
- Edit products
- Remove products
- Upload product images
- Add product price, sizes, category and description
- Change business name, tagline, WhatsApp number, location, social handle and homepage text
- Export saved website data as JSON

IMPORTANT NOTE
This is a front-end/static website admin demo. It stores changes in the same browser using localStorage.
That means it is good for previewing and testing, but it is not a secure live admin system.

For a proper live website where admin changes show for everybody online, connect this design to a backend/database such as Firebase, Supabase, WordPress/WooCommerce, Shopify, Laravel/PHP + MySQL, or Node.js + PostgreSQL/MongoDB.

HOW TO TEST
1. Open index.html.
2. Open admin.html.
3. Login with the default admin details.
4. Add or edit a product.
5. Go back to index.html in the same browser.
6. The product will appear under "Available Products".

HOW TO HOST
Upload all files together to your hosting public folder.
For a static preview, Netlify, Vercel, GitHub Pages, or cPanel hosting can work.
For a real admin system, use backend hosting with a database.
